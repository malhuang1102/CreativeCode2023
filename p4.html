Project #4 Exquisite Corpse

Description

The equisite corpse, a technique invented by surrealists in the 1900s, in which players write in draws on a sheet of paper, fold it to conceal part of the drawing, and then pass it to the next player for a further contribution. This project aims to achieve the same effect, combining parts of a character from three contributor, we generated this unexpected being.

Design Process

Partnering up with Matthew and Yu, we each designed our character first. I wanted my character, to fit the bizarre, surrealist, Frankenstein-style theme of Equisite Corspe, to look like an elusive, stern mystical creature. So, I drew inspiration from the Egyptian Deity drawing and a dark color theme and "mixed and matched" elements into this character.

Original Sketch

However, translating this image into code was not a very breezy process. I had to depend on the SVG to p5 converter heavily to convert the irregular curves to p5 code. Althought it's quite useful translating simple bezier curves, it sucks at translating more complex drawings. The results weren't optimal and required adjusting the placement of each elements, also because I wanted to still preserve the flexibity to change the colors of the drawing, so instead of coding my drawing in top, middle, bottom, three parts, I devided the whole drawing into layers of shapes, then "mask" the hidden parts when only 1/3 is needed to complete the composition.

Reflection

The downside of using the SVG to p5 convert is, I had to use a lot of very specific translate commands to place things where they need to be. This made my functions less flexible when added to other's code - an important reminder to try to design objects in my program more as independent working parts, and always think about how appliable my functions are when more people are buildling the same system.
