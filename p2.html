Project #2 Parameterized Design

Description

For this assignment I wanted to design a small creature that has various expressions and changes its "mood" or "character" with the movement of the mouse.

Design Process

I started by drawing a small square as the face and making it look more rounded or sharp by changing the radius of the filets around the edges. Then I added details of the squinting eyes and snugly smile. To make it feel more interactive I make the face move along with the mouse in the center of the canvas using map().

Reflection

Fun! I'm pretty happy with how the face turned out. I did it incrementally, adding feature one by one so I didn't go back and forth fixing things. Next time I'll challenge myself to use more functions.
